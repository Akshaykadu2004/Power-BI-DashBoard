# 🎧 Spotify Power BI Dashboard

## 📌 Overview
This project is an interactive Spotify dashboard built using Power BI to analyze songs, artists, and music trends.  
It provides insights into song popularity, artist performance, and listening patterns using dynamic visuals.

---

## 📊 Dashboard Features
- 🎵 Songs and Artist panel with album visuals  
- 🎤 Artist Analysis:
  - Artist by number of songs  
  - Artist by total popularity  
  - Artist by chart positions  
- 🎶 Song Analysis:
  - Songs by popularity  
  - Songs by position  
  - Song-level detailed table  
- 📅 Trend Analysis:
  - Average popularity by month  
  - Distinct songs by month and year  
- 📊 KPI Cards:
  - Total distinct songs  
  - Total artists  
  - Average duration  
  - Average popularity  

---

## 🛠️ Tools Used
- Microsoft Power BI  

---

## 📂 Project File
- Spotify Power BI DashBoard.pbix  

---

## 🚀 How to Use
1. Download or clone this repository  
2. Open the `.pbix` file in Power BI Desktop  
3. Navigate between pages (Overview, Artists, Songs)  
4. Use filters and visuals to explore insights  

---

## 📈 Key Insights
- Identification of top artists like Taylor Swift and others  
- Most popular songs based on popularity score  
- Monthly trends in song releases and popularity  
- Distribution of songs by album type (single vs album)  
- Explicit vs non-explicit content comparison
